# DataSight

# Introduction

DataSight, created using [R](https://www.r-project.org/) and [Shiny](https://www.rstudio.com/products/shiny/), is an application designed to allow users to upload datasets in `xlsx`, `text` or `csv` format. This application translates the data into insightful visualizations and conducts statistical analysis.
